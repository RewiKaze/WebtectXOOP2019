START LESSON

ยินดีต้อนรับเหล่านักเรียนทั้งหลาย เข้าสู่ STUDY PLANET โลกแห่งการเรียนรู้การเขียนโปรแกรมเชิงวัตถุด้วยภาษาจาวา โดยเนื้อหาจะประกอบไปด้วยความรู้พื้นฐานของภาษาจาวา ไปจนถึงหลักการเขียนโปรแกรมเชิงวัตถุพื้นฐาน โดยถ้าพร้อมที่จะไปผจญภัยแล้ว ก็ไปเริ่มกันเลย !

ภายในบทเรียนจะประกอบทั้งหมด 5 บทเรียน โดยการที่จะผ่านในแต่ละบทได้นั้น ก็จะมีเงื่อนไข!! ( ไม่ให้ผ่านไปได้ง่ายๆหรอกนะ )

เมื่อเรียนแต่ละบทจบแล้ว จะมีควิซหรือคำถามท้ายบทนั่นแหละให้ได้ทบทวนกัน โดยถ้าหากตอบคำถามได้ถูกต้อง บทเรียนต่อไปก็จะทำการปลดล็อคให้ทันที แต่ถ้าหากไม่ถูกก็สามารถทำได้เรื่อยๆ แต่ขอบอกเลยว่าไม่ยากเกินไปแน่นอน!

ถ้าหากรู้สึกว่าเรียนๆไปแล้วเกิดรู้สึกท้อ หรืออยากมาทำต่อวันหลังก็สามารถกดปุ่ม SAVE เพื่อทำการบันทึกว่าเรียนไปถึงบทเรียนไหนแล้วด้วย !~~ ( แต่อย่าลืม LOGIN ทุกครั้งนะ )

เมื่อไปแอบๆดูหน้าโปรไฟล์ของเราแล้วก็จะสามารถรู้ได้ด้วยว่าเราเรียนไปกี่เปอร์เซนต์ของบทเรียนทั้งหมดแล้วด้วย ~

ถ้าพร้อมแล้วก็ไปเริ่มเรียนกันได้เลย !
