- Lesson 3 : ## ตัวดำเนินการคอมเม็นต์ ( Comment Operator )-
ตัวดำเนินการชนิดนี้เป็นตัวดำเนินการที่มีไว้เพื่ออธิบายโปรแกรม ยกตัวอย่างเช่น
เวลาที่เราไปอ่านโค้ดของเพื่อนแต่ละคนก็จะมีสไตล์การเขียนที่ต่างกัน ซึ่งตัวดำเนินการคอมเม็นต์เนี่ย จะช่วยให้เราเข้าใจโค้ดที่เพื่อนเราเขียนมากขึ้น ( มีไว้กันงงนั่นเอง )

โดยนอกจากนี้ถ้าหากเกิดบัคหรือข้อผิดพลาดของโปรแกรมก็ทำการแก้ไขและปรับปรุงได้สะดวกขึ้น ( เพราะมีตัวอธิบายยังไงละ! )

คอมเม็นต์ในภาษาจาวาถูกแบ่งออกเป็นสามประเภทหลักๆ ดังนี้
```java
//This is a comment #สำหรับบรรทัดเดียว
/* This is also a comment */ #สำหรับข้อความตั้งแต่หนึ่งบรรทัดขึ้นไป
/** This is a comment for documentation */
#สำหรับข้อความที่ต้องการสร้างเป็นไฟล์เอกสารที่เป็นไฟล์ประเภท HTML
```
