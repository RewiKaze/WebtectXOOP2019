Lesson 4 : ตัวดำเนินการคณิตศาสตร์ ( Arithmetic Operators )
ตัวดำเนินการชนิดนี้เป็นตัวดำเนินการพื้นฐานที่ใช้ในการคิดคำนวณทางด้านคณิตศาสตร์พื้นฐานเช่น
บวก ลบ คูณ หาร
โดยรวมไปถึง มอดดูโล (เศษจากการหาร) และเลขยกกำลังอีกด้วย

| เครื่องหมาย | ความหมาย |
|:--------:|:-------------:|
| + | บวก |
| - | ลบ |
| * | คูณ |
| / | หารเอาเศษ |
| // | หารไม่เอาเศษ |
| % | เศษจากการหาร|

ถ้าหากต้องการย่อรูปตัวดำเนินการคณิตศาสตร์ สามารถทำได้ดังนี้
```java
int a = 0;
a += 3; #มีค่าเท่ากับ a = a+3;
```
