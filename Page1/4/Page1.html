บทที่ 4 : OOP Programing

    ยินดีต้อนรับเหล่านักเรียนทั้งหลาย เข้าสู่ STUDY PLANET โลกแห่งการเรียนรู้การเขียนโปรแกรมเชิงวัตถุด้วยภาษาจาวา โดยเนื้อหาจะประกอบไปด้วยมาเรียนรู้เกี่ยวกับการเขียนโปรแกรมเชิงวัตถุ รวมไปถึงเมธอด และคีย์เวิร์ดมีบทเรียนย่อย ดังนี้
    1. การห่อหุ้มข้อมูล ( Encapsulation )
    2. การสืบทอด ( Inheritance )
    3. ความสัมพันธ์ ( Relationship )
    4. การมีได้หลากหลายรูปแบบ ( Polymorphism )
    5. คอนสตรัคเตอร์ ( Constructor )
    6. คีย์เวิร์ด this และ super & this() และ super()
    7. Special parts (1)
โดยถ้าพร้อมที่จะไปผจญภัยแล้ว ก็ไปเริ่มกันเลย !

วิธีการเข้าเรียน

 ภายในบทเรียนจะประกอบไปด้วย 7 บทเรียนย่อย ซึ่งการที่จะผ่านแต่ละตอนไปได้นั้น ก็จะมีควิซให้ได้ทบทวนกัน ถ้าหากตอบถูกก็จะไปบทต่อไปได้ทันที ~ แต่ถ้าผิดก็ทำใหม่ได้อีก บอกเลยว่าไม่ยากเกินไปแน่นอน!

สมมุติว่าหากเรียนๆ แล้วรู้สึกท้อก็มาเรียนต่อวันหลังได้นะ! ( ระบบจะทำการ SAVE ไว้ให้อิอิ )
