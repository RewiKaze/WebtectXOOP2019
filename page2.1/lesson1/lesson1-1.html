บทที่ 1 : สวัสดี, จาวา

ในบทนี้เราจะมาเรียนรู้เกี่ยวกับประวัติความเป็นมาเบื้องต้นของภาษาจาวากัน ว่ามีความเป็นมาอย่างไร รวมไปถึงองค์ประกอบของเทคโนโลยีจาวา และขั้นตอนการทำงานของจาวา เพื่อให้สามารถมองภาพรวมของจาวาได้อย่างง่ายดายขึ้น

ประวัติของภาษาจาวา

ภาษาจาวาถูกพัฒนาขึ้นเพื่อต้องการที่จะควบคุมเครื่องใช้ไฟฟ้าขนาดเล็ก โดยทีมวิจัยของบริษัทซันไมโครซิสเต็มส์ แต่ก่อนนั้นชื่อเก่าของมันก็คือ Oak ก่อนที่จะทำการเปลี่ยนชื่อภาษาเป็น Java ในภายหลังนั่นแหละ

โดยภาษาจาวานั้นเป็นภาษาที่ไม่ขึ้นกับแพลตฟอร์ม ( platform independent ) โดยโปรแกรมที่ทำงานบนแพลตฟอร์มของมันสามารถใช้ภาษาอื่นในการเขียนได้ เช่น Python และมีการประกาศใช้ JDK เวอร์ชั่น 1.0 เมื่อปี 1996 ซึ่งในปัจจุบันคือเวอร์ชั่น 12.0.2

องค์ประกอบของเทคโนโลยีจาวา

โดยองค์ประกอบหลักนั้นสามารถแบ่งได้ 3 ส่วนได้แก่ 1. JDK ( Java Development Kit ) / J2SDK ( Java 2 Software Development Kit ) : เป็นชุดพัฒนาโปรแกรมภาษาจาวา 2. JRE ( Java Runtime Environment ) : มีหน้าที่ในการประมวลผลโปรแกรม 3. JVM ( Java Virtual Machine ) : ทำหน้าที่เป็นอินเตอร์พรีตเตอร์

ขั้นตอนการทำงาน

สามารถแบ่งได้หลักๆ 3 ขั้นตอน ได้แก่

1. เขียน Source Code ใน Text Editor

Example.java


2. Compile โปรแกรม ซึ่งทำให้สามารถรันได้ทุก Platform โดยจะทำการแปลงจาก .java เป็น .class

javac Example.java → Example.class


3. รันโปรแกรม โดยจะได้เป็นไฟล์ .exe ( สามารถนำไปใช้งานได้เลย! )
